# 政策用语标注指南

## 1. 概述

本指南规范政策用语的标注方法，确保系统对政策语言的解读一致、可追溯、不过度延伸。标注内容将用于 `skill/policy_language_lexicon.md` 词库的维护和更新。

## 2. 标注原则

### 2.1 证据优先

所有标注必须有明确的政策文本依据。不得凭主观判断添加不存在的含义。

### 2.2 审慎保守

宁可标注不足，不可标注过度。对于模糊表述，应标注其不确定性，而非强行赋予确定含义。

### 2.3 语境敏感

同一用语在不同文件层级、不同上下文中可能有不同含义。标注时须注明适用语境。

### 2.4 不越级推理

标注结果不得超过原文本的证据等级。如原文为"鼓励"，不得标注为"补贴"。

## 3. 标注字段

每个政策用语标注包含以下字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| 政策用语 | 是 | 原文用语，精确到字 |
| 字面解释 | 是 | 语言学层面的字面含义 |
| 实际含义 | 是 | 在政策语境中的真实含义 |
| 强度级别 | 是 | 高/中高/中/中低/低 |
| 常见含义 | 是 | 最常见的政策信号解读 |
| 注意事项 | 是 | 容易产生的误读 |
| 来源文件 | 是 | 依据的政策文件（至少一个） |
| 文件层级 | 是 | 文件的来源层级（S/A/B/C/D/E） |
| 语境限定 | 否 | 特别适用或不适用的语境 |

## 4. 强度级别判定标准

### 4.1 高

- 配套措施已明确：专项资金、责任部门、完成时限
- 违反有明确法律后果
- 出现在S级或A级文件中且表述明确无歧义
- 示例用语：严禁、必须、坚决、严厉打击、确保

### 4.2 中高

- 政策方向明确，资源投入预期存在
- 执法标准提升
- 需等待实施文件确认具体规模
- 示例用语：大力发展、加大力度、加快、强化、严厉打击（执法层面）

### 4.3 中

- 方向支持但具体措施待定
- 制度建设信号
- 效果取决于配套措施
- 示例用语：推进、促进、支持、完善、规范发展、积极稳妥

### 4.4 中低

- 延续既有方向，不是新变化
- 节奏控制信号
- 灵活性大，具体标准待定
- 示例用语：持续推进、有序推进、稳步、鼓励、稳步推进

### 4.5 低

- 倡导方向，不强制
- 观察阶段，无行动
- 判断空间留给执行者
- 示例用语：探索、引导、适当、密切关注、合理

## 5. 标注工作流

### 5.1 发现新用语

1. 在政策文本中发现未在词库中的用语
2. 收集至少3个不同来源的该用语使用实例
3. 分析不同层级文件中该用语的含义差异
4. 按标注字段填写标注记录

### 5.2 更新已有标注

1. 发现已有标注与实际政策文本不符
2. 收集新的证据
3. 提出修改建议，附证据链
4. 创建PR更新词库

### 5.3 争议处理

1. 如对某个用语的解读存在争议
2. 在标注中添加"语境限定"字段说明争议
3. 同时列出保守解读和积极解读
4. 标注争议未解决状态

## 6. 常见误读模式

以下误读模式须在标注中特别关注并在"注意事项"字段中提醒：

| 模式 | 错误推理 | 正确做法 |
|------|----------|----------|
| 鼓励≠补贴 | "鼓励"推出后必定有补贴 | 标注"鼓励≠补贴"，需等待实施文件确认激励形式 |
| 试点≠全国 | 试点地区的效果可推断为全国效果 | 标注"试点≠全国"，需等待复制推广文件 |
| 支持≠成功 | 政策支持保证行业参与者成功 | 标注"支持≠成功"，市场因素决定实际结果 |
| 打击≠取缔 | 执法行动等于整个行业被禁止 | 标注"打击≠取缔"，执法范围有明确界限 |
| 规划≠实现 | 规划目标等同于已实现结果 | 标注"规划≠实现"，历史达成率不一 |
| 讲话≠政策 | 领导人讲话等于即刻生效的政策 | 标注"讲话≠政策"，从讲话到政策有时间差 |
| 上升≠战争 | 风险上升等于战争不可避免 | 标注"上升≠战争"，风险等级不等于确定结果 |
| 单文件≠转向 | 单一文件代表政策方向变化 | 标注"单文件≠转向"，需多源交汇确认 |
| 地方≠全国 | 地方政策经验可直接在全国复制 | 标注"地方≠全国"，地方条件不可泛化 |
| 沉默≠默许 | 官方未表态等于允许 | 标注"沉默≠默许"，沉默无信息量 |

## 7. 审核标准

### 7.1 标注质量检查

每条标注提交前须通过以下检查：

- [ ] 政策用语原文精确
- [ ] 字面解释客观中立
- [ ] 实际含义有至少一个政策文本支撑
- [ ] 强度级别与判定标准一致
- [ ] 注意事项包含至少一条常见误读
- [ ] 来源文件真实可查
- [ ] 文件层级标注正确

### 7.2 定期审核

- 每月审核一次词库，检查是否有需要更新的条目
- 每季度校验强度级别与实际政策走向的一致性
- 每半年评估误读模式覆盖率
