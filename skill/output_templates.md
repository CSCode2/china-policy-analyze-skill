## Template 8: Anti-overinterpretation Warning

Use this template whenever the analysis detects a risk of overinterpretation, or when attaching to any answer that involves inference beyond the explicit text.

```markdown
## ⚠️ 过度解读警示

### 当前分析中的推断层级

| 推断类型 | 内容 | 证据充分性 |
|---------|------|-----------|
| 原文明确表述 | [逐字引用原文] | 充分 |
| 结构化解读 | [基于文件结构和上下文的解读] | 较充分 |
| 合理推断 | [基于政策阶段和实施工具的推断] | 中等 |
| 延伸推测 | [逻辑上可能但证据不足的推测] | 不足 |

### 可能的过度解读方向

| 过度解读 | 为什么是过度的 | 正确的边界 |
|---------|--------------|-----------|
|          |              |           |

### 常见过度解读模式提醒

1. 将"支持"等同于"必然成功" → 政策支持是必要条件,非充分条件
2. 将"试点"等同于"全国推广" → 试点可能失败或被调整
3. 将"规划目标"等同于"实现结果" → 规划目标可能未达成
4. 将"严厉打击"等同于"全面取缔" → 打击有范围、对象和程序
5. 将单文件解读为方向转变 → 需要多个文件交叉验证
6. 将外国政策变动直接映射国内影响 → 传导路径可能被政策对冲
7. 将"鼓励"等同于"补贴" → 鼓励可能有多种形式,不一定是资金
8. 将短期数据波动等同于趋势 → 需要更多时间窗口确认
9. 将领导人讲话等同于立即生效的政策 → 讲话到政策有过程
10. 将地方经验等同于全国经验 → 地方条件差异大
11. 将市场解读等同于政策意图 → 市场可能有偏见
12. 将缺席表态等同于默许 → 缺席可能有多种原因

### 本分析的可信度边界

**可以确定的**: [基于充分证据的结论]

**有条件成立的**: [需要在特定条件下才能成立的结论]

**不能确定的**: [当前证据不足以支持的结论]

**需要后续验证的**: [需要等待新证据的结论]
```
