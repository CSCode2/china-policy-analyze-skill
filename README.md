## 为什么做这个项目？

中国政策有一个鲜明特点：**方向一旦写进重要文件，往往不会停留在纸面上。** 它会被部委细化，被地方承接，被**资金、项目、土地、税收、信贷、考核**等机制逐步推动落地。

政策语言也不是随便写的。**"鼓励""支持""大力发展""积极稳妥""规范发展""依法严厉打击"**，这些词背后代表的是不同的政策强度、执行节奏和风险边界。

> **《中国制造2025》就是一个例子。** 2015 年，它提出制造强国的十年行动纲领，重点部署新一代信息技术、高端装备、新能源汽车、新材料等方向。十年后，中国在新能源汽车、光伏、工业机器人、5G 工厂、智能制造等领域已经形成了明显优势。很多后来的产业变化，其实早就写在政策蓝图里。

这说明：**研究中国公开政策，是普通人理解未来中国发展方向的一种高性价比方法。**

它不是唯一方法，也不能保证每个机会都成功。但它能提前告诉你：

- **国家重视什么**
- **资源流向哪里**
- **哪些行业会被扶持**
- **哪些风险会被约束**
- **哪些规则正在变化**

这是一种**巨大的公开信息红利**。

---

问题是，**大多数人没有意识到这一点。** 意识到的人，也很难每天跟踪几十个政府网站，读懂大量公文，理解政策语言，再把中央文件、地方政策、产业动态、司法执法、外贸外交和风险信号整合起来。

**China Policy Intelligence Skill 要解决的就是这个问题。**

它持续跟踪权威公开来源，自动采集政策文件、重要讲话、会议通稿、地方落地文件、司法执法案例、外贸外交动态和地缘风险信号。

然后，它把这些材料翻译成普通人能看懂的分析：

- 这句话是什么意思？
- 政策力度有多大？
- 和过去有什么关系？
- 现在处在试点、推广还是监管阶段？
- 会影响哪些行业和人群？
- 哪些判断有依据，哪些不能过度解读？

> **政策不会替你做决定，但它会提前留下方向。能读懂方向的人，往往更早开始准备，更早抓住机会。**

## 效果示例

### 问：十五五规划定了哪些大方向？对我有什么影响？

```
政策分析    ❯ 《中华人民共和国国民经济和社会发展第十五个五年规划纲要》（2026年3月13日，
             全国人大批准），这是未来五年中国发展的总路线图。全文6万字，我挑关键的讲：

             宏观数字：
             - 经济增长：GDP年均增长"5%左右"，到2030年服务业规模突破100万亿元
             - AI产业："十五五"末人工智能相关产业规模增长到10万亿元以上
             - 智能机器人等六大新兴支柱产业产值到2030年有望扩大到十万亿元以上
             - 普通高中增加学位200万个以上，社区养老覆盖率提高到70%以上
             - 制造业+服务业每年创造超过1000万个就业岗位

             对你意味着什么：
             🟢 如果你在AI/机器人行业：这是明确的万亿级赛道信号，国家配资源、配政策
             🟢 如果你在服务业：扩能提质是主线，"100万亿"意味着这个盘子还在快速变大
             🟢 如果你在教育/养老：学位和覆盖率目标 = 硬性KPI = 会真金白银地投入
             🟡 如果你做传统制造业：没有明说"收缩"，但信号重心明显偏向新质生产力

             证据等级：L1（十五五规划纲要原文）- 全国人大 2026-03-13
```

### 问：最近服务业有什么新政策？

```
政策分析    ❯ 《国务院关于推进服务业扩能提质的意见》（国发〔2026〕7号，
             2026年4月14日），这是近年来服务业领域最高规格的政策文件。

             核心信号：
             - 首次提出服务业总规模"迈上100万亿元台阶"的量化目标
             - "扩能"和"提质"并列，意味着不是简单做大，而是优质增量
             - 特别强调"深入实施人工智能+行动"、"加快工业软件创新突破"

             对你意味着什么：
             🔹 如果你做 AI/软件：政策明确"支持采购大模型、智能体服务"，
                这是真金白银的需求信号
             🔹 如果你做物流：多式联运"一单制"落地、铁水联运加快，
                运输成本有下降空间
             🔹 如果你做金融：供应链票据、数字人民币赋能、
                跨境供应链金融标准互认——跨境金融利好

             证据等级：L1（政策原文）- 国务院 2026-04-14
```

### 问：新能源车补贴最近有什么变化？

```
政策分析    ❯ 扫描最近30天的政策文件，发现以下变化：

             📄 财政部 国家税务总局《关于新能源汽车车辆购置税政策的公告》
                （2026年4月2日）
                - 继续免征，但标准从"纯电续航"转向"单位载质量能耗"
                - 影响：低端微型电车可能不再享受优惠

             📄 工信部《道路机动车辆生产企业及产品》新批次公告
                （2026年3月28日）
                - 新增 47 款新能源车型
                - 固态电池车型首次出现在公告中

             📄 发改委 产业司座谈会议（2026年4月15日）
                - 信号词："优化补贴机制""由补购买转向补使用"
                - 尚属会议信号（L4级），未形成正式文件

             证据等级：L1 / L3 / L4 - 已区分事实、推断和不确定
```

### 问：地缘冲突风险怎么样？

```
政策分析    ❯ 本周地缘风险信号扫描：

             风险上升信号：
             ▲ 美国BIS新增12家中国实体进入出口管制清单
                （2026年4月22日，L1/OFAC官网）
             ▲ 东部战区绕台巡航频次较上月增加30%
                （2026年4月20日，L2/国防部发布）
             ▲ USTR启动对华海事物流301调查新阶段
                （2026年4月18日，L1/USTR官网）

             风险平缓信号：
             ▼ 中美商务部长通话，同意保持沟通渠道
                （2026年4月19日，L2/商务部）
             ▼ 南海局势未出现新的对峙事件
                （至2026年4月27日，L3/海警局周报）

             注意：我不做战争日期预测，也不做"某月一定开战"的判断。
             上述仅为信号梳理，你可以据此自行评估。
```

> 完整每日报告在 `reports/` 目录下自动生成。

这不是 ChatGPT 套了个政策分析面具。每条分析都在标注证据等级——L1 原文、L2 官方解读、L3 媒体报道、L4 会议信号、L5 专家观点——它不混为一谈，你一眼能看清哪个是事实、哪个是推断。

## 蒸馏了什么

### 7种结构化卡片

| 卡片类型 | 做什么 | 举个例子 |
|----------|--------|----------|
| **文件卡** | 一个政策文件的结构化摘要 | 《国发〔2026〕7号》- 服务业扩能提质意见 |
| **政策信号卡** | 提取政策中的关键变化信号 | "由补购买转向补使用" - 新能源车补贴转向 |
| **政策语言卡** | 解码政策术语的真实含义 | "稳妥推进" = 方向确定但步子慢 |
| **地方落地卡** | 中央政策在各省市的落地差异 | 同一政策，浙江vs河南不同执行方案 |
| **机会卡** | 政策扶持带来的潜在机会 | AI+行动 → 大模型/智能体采购需求 |
| **案例卡** | 与政策相关的司法/执法案例 | 反垄断处罚案例库 |
| **概念卡** | 政策高频概念解释 | "新质生产力" "全国统一大市场" |

### 5级证据等级

| 等级 | 含义 | 可信度 |
|------|------|--------|
| **L1** | 政策原文引用 | 最高 |
| **L2** | 官方解读/配套文件 | 高 |
| **L3** | 官媒报道 | 中高 |
| **L4** | 会议信号/领导讲话暗示 | 中 |
| **L5** | 专家研判/市场推断 | 参考用 |

### 政策语言词典（节选）

| 政策用语 | 实际含义 |
|----------|----------|
| "深入研究" | 还没开始做 |
| "稳妥推进" | 方向确定但步子慢 |
| "加快" | 真的要快了 |
| "大力推进" | 资源已经到位了 |
| "毫不动摇" | 这事有人反对，但中央定了 |
| "分类推进" | 不是一刀切，各地不同 |
| "视情开展" | 先看看再说 |
| "严禁" | 真的禁了 |

完整词典见 [`skill/policy_language_lexicon.md`](skill/policy_language_lexicon.md)

## 它不能做什么？

老实说在前头：

- **不给投资建议。** 政策扶持不等于股票必涨，机会分析一定带着风险提示。
- **不做确定性预测。** "某年某月一定会发生某事"这种话我们不说，也反对别人说。
- **不碰涉密和军事内容。** 这是红线。
- **不替你做决策。** 工具提供信息，决策永远是你自己的事。
- **不做制裁规避建议。** 监测制裁信息是为了帮你了解风险，不是教你绕开它。

## 实在想看英文？

China Policy Analyze Skill is an open-source system that automates policy text collection, structural interpretation, scenario assessment, and early warning from 120+ authoritative Chinese government sources. It turns dense policy documents into actionable daily briefings with evidence-level annotations (L1–L5), multi-scenario analysis, and 7 types of structured distillation cards.

## 安装

### 环境要求

- Python 3.10+
- pip 或 uv

```bash
git clone https://github.com/CSCode2/china-policy-analyze-skill.git
cd china-policy-analyze-skill

# 核心依赖
pip install -e .

# 开发依赖（可选）
pip install -e ".[dev]"
```

### 配置

```bash
cp .env.example .env
# 编辑 .env 填入你的 API 配置
```

### 运行

```bash
# 每日更新（采集+解析+报告）
source venv/bin/activate
CPI_MAX_DOCS=10 python scripts/_run_daily_update.py

# 运行评估测试
python -m pytest tests/ -q
```

## 核心理念
